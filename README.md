Pet simulator

Pet simulator is a Rest API for simulating a tamagotchi-like pet. can you feed, play with, and take care of your pet through various endpoints.
Also you can customize his actions and behaviors.

## Setup:
The config file is located in `config.toml`. You can modify the settings there to fit your needs.

## Architecture:
The project is built using Python and FastAPI for the backend. It uses SQLite as the database to store pet data.
Also, the project focuses on modularity and scalability, allowing for easy addition or modification of any services or features always using abstraction layers for communication between services modules.
